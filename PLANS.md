
## User Stories
### Basic
```
As a developer
So that I know that my code meets quality standards
I want to add coverage, code quality, code style checks
```
```
As the client
So that I can view progress
I want the app deployed on Heroku
```
```
As a developer
So that my working product is automatically updated
I want to set up continuous integration and deployment
```
```
As a user
So I can begin posting to Instagram
I want to be able to sign up

Additional features as part of this user story:
   Profile photos with Gravatar
   only valid emails should be allowed
   Can sign-up and sign-in with Facebook
```
```
As a user 
So that I can access my profile and post
I want to be able to sign in
```
```
As a user
So that I can share my photos with the world
I want to be able to post photos to Instagram with a caption, and see them displayed

Advanced features that can be added:
   Username, profile photo, and date/time of posting is displayed next to each post
   Line breaks in captions are preserved
   Users can also add videos
```
```
As a user
So that I can interact with other users on the platform
I want to be able to like and comment on photos

Features:
   A post or comment can only be liked once by each user 
```
### Bonus
```
As a user
So that I can remove photos I no longer like or want online
I want to be able to delete photos

Feature: 
   Should only be able to delete their own photos, and receive a helpful message if attempting to delete others users' posts
```
```
As a user
So that I can change my original photo caption
I want to be able to edit my posts

Feature:
   Should only be able to edit their own photos, and receive a helpful message if attempting to edit other users' posts

Additional features that can be added:
   Filter can also be changed when editing
```
```
As a user
So that I can make my photos prettier when I post them
I want to be able to use filters
```
```
As a user
So that I can see the most relevant photos to me
I want the posts to be shown from newest to oldest
```
```
As a user
So that I can interact with other users on the site
I want to be able to follow other users and have them follow me
```
```
As a user
So that my experience is smooth
I want the site to be interactive and and nice to look at

Features:
   Home, profile, new post buttons always show in navbar
   Photos enlarge when clicked on

Advanced:
   Implement React
```
### Advanced
```
As a user
So that I can see my profile
I want a dedicated profile page that can be easily accessed, and shows my posts

Additional features that can be added:
   User profile shows follower and following count
   Clicking on the user icon next to a post takes you to that user's profile page
   Clicking on followers or following shows the respective list, with avatars and usernames
   You can click on those usernames and go to their profiles too	
```
```
As a user 
So that I can chat to other users
I want to be able to message people privately

Features:
   Multiple private chat rooms
   Multiple chat users per room (can be 1-1)

Advanced features:
   Online / Offline status of each user
   Real-time “typing…” status
   Real-time read receipt
```
